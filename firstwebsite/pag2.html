O que é o CSS

O CSS, ou Cascade Style Sheets, é uma linguagem de descrição de estilos que define o layout de documentos HTML. O CSS controla a formatação e texto e posicionamento dos elementos através de parâmetros. Enquanto o HTML define o conteúdo da página, o CSS formata e estiliza o conteúdo.

A sintaxe do CSS

Em CSS, a sintaxe é muito simples e consiste em: 

					seletor {
					  propriedade: valor;
					}

O seletor

O seletor representa uma estrutura que determina quais elementos dentro de um grupo de elementos será formatado. É possível encadear seletores de forma a especificar o elemento desejado. 

					.menu p a {
					  font-size: 16px;
					}
O código acima seleciona o link a, contido no parágrafo p, dentro da classe menu e altera o tamanho da fonte.

Sintaxe simples

Em resumo, a sintaxe do CSS é composta por uma descrição de propriedades separados por ":" e finalizado com ";"

As três implementações

Existem 3 maneiras de se implementar o CSS em uma página HTML:

  • Inline
  • Incorporada
  • Importada

Inline: atributo style declarado no próprio elemento HTML. 

					<p style="color: #000; background-color: #111;">

Incorporada: Tag style e uso de seletores dentro do arquivo HTML: 

					<style type="estilocss">
					h1 {
					  color: #000;
					}

					p {
				  	  font-family: monospace;
					  color: #111;
					}
					</style>

Importada: Implementação do CSS em arquivo externo, que é, então, importado ao HTML correspondente. Essa importação ocorre dentro do <head>, da seguinte forma: 

					<link rel="stylesheet" href="main.css">
