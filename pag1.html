Tags do HTML

  • <head>: Contem os metadados da página.
  • <body>: Contem o conteúdo da página.
  • <h1>, <h2>, <h3>... <h6>: Representam os títulos de cabeçalhos do documento. Um elemento título descreve o tema da seção.
  • <p>: Indica uma parte do texto que deve ser exibida como parágrafo.
  • <pre>: Indica texto pré formatado cuja formatação deve ser preservada.
  • <blockquote>: Representa uma citação.
    • Exemplo:
    Quem confere ferro com ferro será conferido.
  • <ol>: Define uma lista ordenada, podendo indicar valor inicial.
    • Exemplo:
    1. Item 1
    2. Item 2
  • <ul>: Define uma lista não ordenada.
    • Exemplo: Esta própria lista :D
  • <li>: Define um item da lista ordenada ou não ordenada.
  • <dl>: Representa uma lista de definição, onde dt cria um item e dd define o item.
    • Exemplo:
    Item 1
    Definição do item 1
    Item 2
    Definição do item 2
  • <a href="pagina.html">: Define hipertexto para outra página dentro do site.
    • Exemplo: Clica ai!
  • <code>: Define inserção de código.
    • Exemplo: for(temp = lista ;temp != NULL; temp = temp->prox)
  • <div>: Define um contêiner genérico sem significado especial.

Tag vs Elemento

A diferença entre tag e elemento em HTML é que a tag é o conjunto de caracteres que formam um elemento, ao passo que, elemento se refere ao conjunto todo. Uma tag apenas marca o início e o fim de um elemento.

Inserção de Imagens

A inserção de imagens em HTML consiste no uso da tag <a href="caminho/imagem.png"> criando hiperlink para a imagem e a tag img com os atributos src(url da imagem ou caminho no pc), height="x"(altura), width="x"(largura), dentre outras tags.
