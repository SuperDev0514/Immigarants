JavaScript

O JavaScript é a linguagem responsável por conceder ação à uma aplicação web. Enquanto o HTML determina quais elementos serão dispostos e a ordem em que serão dispostos e o CSS determina suas propriedades. O JavaScript interage com esses elementos concendendo maior interação através de funções, variáveis e logical statements assim como qualquer linguagem estruturada.

A tag <script>

Em HTML, scripts devem ser inseridos ao redor da tag script (necessita fechamento).

Funções em JavaScript

Uma função em JS é um bloco de código que pode ser invocado no HTML, por exemplo, após um clique em um botão.

Scripts inline ou externos

Um script pode ser feito de maneira inline dentro da tag script ou invocado de um arquivo .js, sendo este mais vantajoso visto que separa do HTML o código em JS fazendo com que ambos fiquem mais limpos e fáceis de ler. Além disso com o arquivo externo fica mais fácil chamar a mesma função várias vezes e em páginas diferentes. Exemplo de inclusão de um script: 

            <script src="meuScript.js"></script>
Caso necessário, incluir o caminho até o arquivo JS.

Exemplo de script onmouseover

Através do parâmetro onmouseover da tag <img> e onmouseout, foram chamadas duas funções JS que alteram o elemento de id memejs, seguindo o seguinte código:

<img id="memejs" onmouseover="changeMeme(this)" onmouseout="changeBack(this)" src="ibagens/resident.png"/>

E scripts: 
            function changeMeme() {
                document.getElementById("memejs").src = "ibagens/pogchamp.png";
            }

            function changeBack() {
                document.getElementById("memejs").src = "ibagens/resident.png";
            }

O JavaScript é capaz de alterar, exibir ou ocultar o CSS. Além disso, existem 3 maneiras de exibir dados via JS:

  • Escrevendo em um HTML: innerHTML
  • Escrevendo em uma caixa de alerta: window.alert()
  • Escrevendo no console do navegador: console.log()
